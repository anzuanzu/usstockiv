# US Stock IV

在無法取得選擇權 IV 時，以免費美股日線資料計算 2 至 6 個月的歷史波動率代理值，作為 FCN 條件研究工具。

## 使用方式

1. 免費申請一組 Twelve Data API Key。
2. 開啟網站，輸入美股代碼、FCN 保守加成與 API Key。
3. 按「載入波動率」。
4. 系統會使用對應的 42、63、84、105、126 個交易日視窗，分別顯示 2 至 6 個月的結果。

API Key 只會暫存於目前瀏覽器分頁的 `sessionStorage`，關閉分頁就會移除；不會提交到 GitHub。

## 資料限制

- 使用 Twelve Data 的免費日線 API，資料可用性、免費額度與更新時間受供應商限制；網站不儲存全市場資料。
- EWMA 年化波動對近期日報酬給予較高權重；下跌年化波動只納入負報酬日。
- FCN 保守代理值 = `max(EWMA 年化波動, 下跌年化波動) × (1 + 保守加成)`。預設加成為 15%，可依你的券商報價經驗調整。
- 此方法不涵蓋選擇權波動率偏斜、事件風險、離散股利、流動性或發行人避險成本。

正式 FCN 報價請以發行券商提供的條件與價格補充文件為準。
